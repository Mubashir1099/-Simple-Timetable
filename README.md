# Simple Timetable

A weekly school timetable page showing time slots across Monday–Friday.

## Features
- Table layout with time slots and weekdays
- Styled header row and alternating row colors
- Hover highlight on table cells

## Tech
- HTML5 (`table`, `fieldset`)
- CSS3 (custom table styling, moved out of inline styles into a stylesheet)

## What I practiced
- Structuring tabular data with semantic HTML
- Styling tables (borders, spacing, hover states) with CSS
